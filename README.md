# Система для поиска маршрутов на графах дорог с использованием различных алгоритмов.

## Алгоритмы
A* (однонаправленный)

Алгоритм Дейкстры

Двунаправленный A* (2A*)

Многопоточный двунаправленный A*

## Особенности
Точное вычисление расстояния по гаверсинусной формуле

Работа с графами в формате GeoJSON

Кэширование графа - граф загружается единожды при запуске

## Формат данных 
Формат графа
Формат GeoJSON с дорожной сетью
``` json
json
{
  "type": "FeatureCollection",
  "features": [
    {
      "type": "Feature",
      "geometry": {
        "type": "LineString",
        "coordinates": [[37.615555, 55.752220], [37.615600, 55.752300]]// Рёбра графа
      }
    }
  ]
}
```

## Сборка и запуск
``` bash
cd RoadMapAlgorithms
mkdir build && cd build
cmake ..
make
./pathfinding
```


# Система для решения головоломки "Пятнашки" с использованием различных алгоритмов поиска.

## Алгоритмы
A* с улучшенной эвристикой

Двунаправленный A* с адаптивной эвристикой

Поиск в ширину (BFS)

Двунаправленный BFS

Многопоточные версии BFS и A*

## Особенности
Эффективное кодирование состояния в 64-битное число

Эвристическая функция, оценивающая запутанность пятнашки

Тестирующая система для генерации случайных состояний

## Сборка и запуск
``` bash
g++ FifteesSolverTestSystem.cpp -o fifteesTestingSystem
./fifteesTestingSystem
```
